    Detailed Description

    Component displaying the trip event data in a non editable way. Added buttons allow the user to open a new page to edit the data or delete the event.

    Is intended to be used as a delegate.

    Property Documentation

    [required] comments : string

    User added comments for this event


    [required] cost : int

    The cost of the event, in currency


    [required] costStatus : string

    Indicates whether this event has been paid or not


    currency : string

    The currency in which the cost is set


    [required] endDate : date

    The end date of the event. Both date and time data are used


    [required] endLocation : string

    The location where the event ends


    [required] eventId : int

    The id of the event being displayed.


    [required] operator : string

    The operator of the event


    [required] startDate : date

    The start date of the event. Both date and time data are used


    [required] startLocation : string

    The location where the event starts


    [required] status : string

    The status of this event. This controls the background color of this component


    [required] type : string

    The type of the event being displayed. The appearance of EventOverview is changed according to this value
