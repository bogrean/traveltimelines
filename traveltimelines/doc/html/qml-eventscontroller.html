    Detailed Description

    Defines all actions that can be perfomed on trip events. Anyone wishing to implement those actions must implement the specific handlers.

    Signal Documentation

    createEvent(var newEvent)

    Use this signal for notifications about new trip events being created.

    newEvent contains the details of the new trip event

    Note: The corresponding handler is onCreateEvent.


    deleteEvent(int eventId)

    Use this signal for notifications about removal of an existing event.

    eventId specifies the vent to be deleted.

    Note: The corresponding handler is onDeleteEvent.


    editEvent(var existingEvent)

    Use this signal for notifications about changing the contents of a trip event

    existingEvent contains the complete trip event data, updated

    Note: The corresponding handler is onEditEvent.


    reloadEvents()

    Use this signal to request a fresh reload of the event data.

    Note: The corresponding handler is onReloadEvents.
