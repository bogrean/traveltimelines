    Detailed Description

    A basic application page that allows the user to edit event data:

    • Event type
    • Event start date & time
    • Event start location
    • Event end date & time
    • Event end location
    • Event status
    • Event cost
    • Whether the event was paid or not
    • operator of the event
    • User comments

    Property Documentation

    [required] dispatcher : EventsController

    The dispatcher used to send notifications regarding events


    tripEvent : var

    An object containing the event data


    tripId : int

    The id of the trip the tripEvent belongs to
