    Detailed Description

    Defines all actions that can be perfomed on trips. Anyone wishing to implement those actions must implement the specific handlers.

    Signal Documentation

    createTrip(var newTrip)

    Use this signal for notifications about new trips being created.

    newTrip contains the details of the new trip

    Note: The corresponding handler is onCreateTrip.


    deleteTrip(int tripId)

    Use this signal for notifications about removal of an existing trip.

    tripId specifies the trip to be deleted.

    Note: The corresponding handler is onDeleteTrip.


    editTrip(var existingTrip)

    Use this signal for notifications about changing the contents of a trip

    existingTrip contains the complete trip data, updated

    Note: The corresponding handler is onEditTrip.


    fetchTripData(int tripId)

    Use this signal to request a fresh reload of the events for the trip identified by tripId

    Note: The corresponding handler is onFetchTripData.


    reloadTrips()

    Use this signal to request a fresh reload of the trip data.

    Note: The corresponding handler is onReloadTrips.
